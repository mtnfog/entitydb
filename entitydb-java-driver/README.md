# EntityDB Java Driver

This driver is a client for [EntityDB](https://github.com/mtnfog/entitydb).

## Build

To build the driver:

```
git clone https://github.com/mtnfog/entitydb-java-driver.git
cd entitydb-java-driver
mvn clean install
```
